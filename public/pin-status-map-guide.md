# 太陽光訪問マップ

`public/pin-status-map.html` は、航空写真を見ながら太陽光パネルがある家へピンを立て、訪問結果を共有管理する画面です。

## できること

- 航空写真上でピン追加モードにして家をタップする
- 住所・地名検索後に地図中心へピンを追加する
- 航空写真上に地名・道路名を重ねて表示する
- 住所、駅名、町名などで地図を検索する
- KMLまたはCSVを読み込む
- 地図上のピンをタップする
- 選択中のピンだけ削除する
- アポ、インキ、ドアキ、アプ、再訪、対象外、留守、禁止に変更する
- 変更後のCSVを書き出す
- 担当者とメモを残す
- サーバー起動時は全員で同じピン情報を編集する

Google マイマップの既存データを使う場合は、マイマップからKMLを書き出して、この画面で読み込みます。

## 使う色

- 未訪問: グレー
- アポ: 緑
- インキ: 青
- ドアキ: 紫
- アプ: 黄
- 再訪: オレンジ
- 対象外: 黒
- 留守: 赤
- 禁止: グレー

スター付き、保存済みの場所、行ってみたいは使いません。

## 共有して使う場合

キャリア通信のスマートフォンからも使うには、アプリをインターネット上のサーバーに置き、HTTPSのURLで開きます。

`.env` は次のように設定します。

```bash
HOST=0.0.0.0
ALLOW_REMOTE_ACCESS=true
DATA_DIR=/var/data
SHARE_PASSWORD=共有パスワード
SESSION_SECRET=長いランダム文字列
GOOGLE_MAPS_BROWSER_KEY=Google Mapsのブラウザ用APIキー
```

共有データはサーバーの `data/pin-status-places.json` または `DATA_DIR/pin-status-places.json` に保存されます。公開URLと共有パスワードを知っている人は編集できるため、公開範囲に注意してください。

## 注意

Google マイマップ自体を直接更新する画面ではありません。変更後のCSVをマイマップへ再読み込みするか、この画面をメインの管理画面として使います。
全消去ボタンは誤操作防止のため画面から外しています。
